സംസ്കൃതസന്ധിയെ അടിസ്ഥാനപ്പെടുത്തി 

1) ദീർഘസന്ധി
ഹിമാ + ആലയം = ഹിമാലയം 
വിദ്യാ + അർത്ഥി = വിദ്യാർത്ഥി 
കവീ + ഇന്ദ്രൻ = കവീന്ദ്രൻ 
ശ്രീ + ഈശൻ = ശ്രീശൻ 
ഭാനു + ഉദയം = ഭാനൂദയം
വധൂ + ഉദയം = വധൂദയം

2) ഗുണസന്ധി 
സുര + ഈശൻ = സുരേശൻ
രമാ + ഈശൻ = രമേശൻ
സൂര്യാ + ഉദയം = സൂര്യോദയം 
മഹാ + ഋഷി = മഹർഷി 

3) വൃദ്ധി സന്ധി 
മമാ + ഏക = മമൈക 
ദേവാ + ഐശ്വര്യം = ദേവൈശ്വര്യം
ഉഷ്ണാ + ഓദനം = ഉഷ്ണൌദനം
മഹാ + ഔഷധം = മഹൌഷധം
ഉപ + ഋച്ഛിതി = ഉപാർച്ഛിതി

4)യൺ സന്ധി 
ഇതി + ആദി = ഇത്യാദി
വധൂ + ആഗതാ = വധ്വാഗതാ
പിതൃ + ആജ്ഞാ = പിത്രാജ്ഞാ

5) അയാദി സന്ധി 
നൌ + അക = നായക
